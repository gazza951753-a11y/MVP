# StudyAssist Intel System

MVP-система для поиска площадок/конкурентов, сбора упоминаний, классификации, скоринга и постановки задач оператору.

## Что исправлено в этой версии
- Интерфейс полностью на русском.
- Реальный поиск по интернету (Рунет): DuckDuckGo + fallback Bing.
- В GUI добавлены кнопки:
  - **Запустить поиск по Рунету**
  - **Очистить демо-данные**
- После запуска теперь показывается отчёт: сколько найдено площадок/упоминаний/задач.

## Быстрый запуск (Windows 10, без Docker)
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -e .[dev]
copy .env.example .env
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

Открыть: `http://127.0.0.1:8000/`

## Если видите только 1 строку
1. Нажмите **Очистить демо-данные**.
2. Нажмите **Запустить поиск по Рунету**.
3. Вверху появится инфо-строка с числами найденных данных.

Если снова 0/мало результатов:
- проверьте интернет и firewall/антивирус;
- в `.env` расширьте `DISCOVERY_QUERIES`;
- попробуйте запустить ещё раз через 1-2 минуты.

## Как работает поиск
- `RussianWebCollector` делает online-запросы по RU-тематике и парсит поисковую выдачу.
- Источники: DuckDuckGo HTML + fallback Bing.
- Результаты попадают в `platforms`, `mentions`, затем создаются задачи.

## Настройки (.env)
- `USE_MOCK_COLLECTOR=false` — только реальные сетевые данные.
- `DISCOVERY_QUERIES=[...]` — список ваших поисковых запросов по теме.
