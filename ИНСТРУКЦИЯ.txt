Что делает эта программа?
--------------------------
Программа автоматически ищет в интернете студентов,
которые просят помощь с курсовыми, дипломами, рефератами.
Она заходит в публичные Telegram-каналы и собирает посты,
где студенты пишут "срочно нужна помощь", "дедлайн завтра"
и т.п. Все найденные упоминания сохраняются в базу данных,
и по ним создаются задачи для оператора.


ПЕРВЫЙ ЗАПУСК (делается один раз)
----------------------------------
1. Дважды кликни на файл УСТАНОВКА.bat
   - Подождёт 1-3 минуты пока скачает нужные пакеты
   - В конце напишет "Готово! Теперь запускайте ЗАПУСК.bat"

2. Дважды кликни на файл ЗАПУСК.bat
   - Откроется чёрное окно - это сервер (не закрывай его!)
   - Через 3 секунды автоматически откроется браузер
   - Адрес: http://localhost:8000


КАК ПОЛЬЗОВАТЬСЯ КАЖДЫЙ ДЕНЬ
------------------------------
1. Запусти ЗАПУСК.bat (чёрное окно должно быть открыто)
2. В браузере зайди на http://localhost:8000

В левом меню есть разделы:

  ДАШБОРД - главная страница с итоговой статистикой

  ПЛОЩАДКИ - список найденных Telegram-каналов и групп
    - Кнопка "Запустить" - полный обход всех источников
      (занимает 5-10 минут, работает в фоне)
    - Кнопка "Триггер-скан" - быстрая проверка
      (занимает 1-2 минуты, работает в фоне)
    После нажатия страница обновится сразу, а данные
    появятся через 1-2 минуты - нажми F5 чтобы обновить.

  ЗАДАЧИ - список постов где студенты просят помощь
    - Каждая строка = один пост студента
    - Зелёный цвет = хорошая возможность ответить
    - Красный цвет = риск бана, осторожно
    - Можно менять статус задачи (новая / в работе / готово)

  УПОМИНАНИЯ - все найденные посты с текстом

  НАСТРОЙКИ - добавить свои Telegram-каналы для слежки


КАК ДОБАВИТЬ СВОИ TELEGRAM-КАНАЛЫ
------------------------------------
1. Зайди в "Настройки"
2. Внизу страницы найди раздел "Telegram каналы"
3. Впиши имена каналов - по одному на строку, без символа @
   Пример:
     studizba
     kursovik_help
     moj_kanal
4. Нажми "Сохранить настройки"
5. Нажми "Триггер-скан" в разделе "Площадки"

ВАЖНО: канал должен быть публичным! Если зайти на
https://t.me/s/ИМЯ_КАНАЛА и там видно посты - значит
программа тоже сможет его читать.


ПОЧЕМУ РЕЗУЛЬТАТОВ МАЛО?
--------------------------
Вероятные причины:

1. Каналы из списка по умолчанию могут не существовать
   или не иметь постов про студентов.
   -> Добавь свои каналы в Настройки (см. выше)

2. В каналах нет постов, которые совпадают с триггерами.
   Программа ищет только посты где есть слова:
   "срочно", "дедлайн", "нужна помощь", "антиплагиат",
   "курсовая", "ВКР", "правки", "научрук" и т.д.

3. Нажал кнопку но не подождал.
   -> Подожди 1-2 минуты и обнови страницу (F5)


ОСТАНОВКА ПРОГРАММЫ
--------------------
Закрой чёрное окно ЗАПУСК.bat. Это останавливает сервер.
Все данные сохраняются в файле studyassist.db - они не
удалятся при следующем запуске.


ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК
--------------------------
- Если браузер не открылся - зайди вручную: http://localhost:8000
- Если чёрное окно закрылось само - перезапусти ЗАПУСК.bat
- Если база данных сломалась - удали файл studyassist.db
  и запусти ЗАПУСК.bat заново (данные сбросятся)
- Если пишет "порт занят" - перезагрузи компьютер
